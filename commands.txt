git init	- Створює новий локальний Git-репозиторій у поточній папці.
git clone <url>	- Копіює віддалений репозиторій на ваш комп'ютер.
git status	- Показує стан файлів і репозиторію (змінені, нові, підготовлені до коміту файли).
git add <файл>	- Додає файл до області підготовки (staging area) перед комітом.
git add . -	Додає всі змінені та нові файли до staging area.
git commit -m "повідомлення" -	Зберігає зміни в історії репозиторію з коментарем.
git log	- Відображає історію комітів проєкту.
git branch	- Показує список гілок або створює нову гілку.
git checkout <гілка> -	Перемикається на вказану гілку.
git push origin <гілка>	- Відправляє локальні коміти до віддаленого репозиторію (GitHub, GitLab тощо).
