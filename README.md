Навчальний Python-проєкт

Опис проєкту

Цей проєкт створений для виконання навчального завдання з використанням системи контролю версій Git та платформи GitHub. У межах проєкту будуть реалізовані базові функції роботи з Python-програмами та організація проєктної структури.

Що буде зроблено:
 - Створення локального Git-репозиторію.
 - Ініціалізація та налаштування Git.
 - Створення структури проєкту.
 - Розробка програмного коду на Python.
 - Додавання та фіксація змін за допомогою комітів.
 - Публікація проєкту на GitHub.
 - Документування проєкту у файлі README.md.
